# waveform_predictor model
本研究提出了一种基于扩张卷积神经网络（Dilated CNN）的地震波形预测模型，旨在从1秒的三分量地震波形（100 Hz采样）直接预测未来3秒的完整三分量波形。(该模型使用间断数据进行训练,适合进行间断推理,如只在触发预测条件的时候进行预测,对前后输入的波形连续性要求更低)

模型采用全卷积编码器架构，包含7层扩张卷积残差块与多尺度频率特征提取分支，将三维输入压缩为潜在特征表示，再通过全连接层直接生成900维的三分量预测波形。模型总参数量约为137万，在保持较高预测精度的同时，具备部署到边缘设备的潜力。

在包含61个地震台站的测试集上，预测Z分量的皮尔逊相关系数中位数为0.52，正相关率达到72%。在STA/LTA触发分类任务中，F1分数达到0.96，可准确识别震动事件。此外，该架构可作为基线模型，为后续针对低功耗场景的模型压缩（如输入/输出维度缩减、权重剪枝、INT8量化及纯C推理部署）提供参考。

该模型适用于实时地震预警、LoRa通信延迟补偿及嵌入式智能监测等应用场景。

测试数据来自IRIS（Incorporated Research Institutions for Seismology）全球地震台网，包含IU、US、G、GE四个台网的11个地震台站，共185个连续波形文件。每个文件包含BHZ、BHN、BHE三分量，采样率100 Hz，时长15分钟。测试样本通过滑动窗口方式从连续波形中截取，每段输入1秒（100采样点），预测输出3秒（300采样点）。各台站均匀采样，每个台站最多选取50个测试样本。
